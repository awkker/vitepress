# 编程入门演示

::: tip 使用说明
- 点击幻灯片可以获得焦点
- 使用键盘方向键 ← → 或空格键切换幻灯片
- 点击左下角按钮或按 `F` 键进入全屏模式
- 按 `O` 键查看大纲
- 按 `D` 键切换暗色模式
:::
