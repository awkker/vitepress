## 如何创建新的幻灯片

1. 在 `slides/` 目录下创建新的 `.md` 文件
2. 使用 Slidev 的 Markdown 语法编写幻灯片内容
3. 在 `package.json` 中添加对应的启动脚本
4. 在本页面添加新的 iframe 嵌入

### Slidev 基本语法

```markdown
---
theme: default
title: 我的幻灯片
---

# 第一页

这是第一页内容

---

# 第二页

- 列表项 1
- 列表项 2

---

# 代码展示

\`\`\`python
print("Hello World!")
\`\`\`
```
