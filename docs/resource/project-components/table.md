# Table 表格增强

项目内置了 markdown 原生表格增强能力，无需额外引入组件。

- 自动包裹滚动容器，超出正文可左右滑动。
- 表格后方提供 `复制 Markdown`、`复制 HTML`、`复制样式 HTML` 三个按钮。
- 支持 `:` 与 `=` 的合并语法，以及空单元格自动合并规则。

## 基础语法（真实渲染）

| 组件 | 功能 | 备注 |
| --- | --- | --- |
| Code | 代码高亮、复制、行号 | 支持路径与语言标签 |
| FileTree | 目录结构展示 | 支持文件/文件夹链接 |
| Toast | 全局反馈提示 | 默认 5s 自动关闭 |

```md
| 组件 | 功能 | 备注 |
| --- | --- | --- |
| Code | 代码高亮、复制 | 支持路径与语言标签 |
| Toast | 反馈提示 | 右下角堆叠 |
```

## 合并单元格（真实渲染）

- `:N` 放在首行首列：合并后续 `N` 列。
- `=N` 放在某行首列：合并该行后续 `N` 列。

| 分组 | B | C | D |
| --- | --- | --- | --- |
| :3 | 核心功能组 |  |  |
| A | 登录 | 搜索 | 统计 |
| =2 | 归档任务 | 数据备份 | 低频执行 |
| E | 导出 | 分享 | 推送 |

## 自动合并规则

- 若同一列首行有内容，后续至少两行为空，会自动纵向合并。
- 若同一行首列有内容，后续至少两列为空，会自动横向合并。

|  |  |  |
| 部署说明 |  |  |
| 开发阶段 | 编码 | 联调 |

## 复制说明

- `复制 Markdown`：导出当前表格内容为 markdown 源码。
- `复制 HTML`：导出当前表格的 HTML 结构代码。
- `复制样式 HTML`：导出包含内联样式的 HTML（粘贴后保留完整边框和表格样式）。
