# 项目组件总览

本项目已内置并全局注册以下组件，可在 Markdown 直接使用，无需在页面中单独 `import`。

<Aside type="tip" title="命名规范">
  组件名区分大小写，建议统一使用 PascalCase，例如 <code>&lt;LinkCard /&gt;</code>。
</Aside>

## 快速开始

```md
<LinkCard title="Markdown 创作" href="/guides/index.html" />
```

## 推荐阅读顺序

1. 先看 [Link Cards](./project-components/link-cards) 与 [Asides](./project-components/asides)（页面信息组织）。
2. 再看 [Code](./project-components/code) 与 [File Tree](./project-components/file-tree)（技术文档展示）。
3. 最后看 [Accordion](./project-components/accordion) 与 [Tabs](./project-components/tabs)（交互容器）。
4. 表单与反馈请看 [Checkbox](./project-components/checkbox)、[Checkbox Group](./project-components/checkbox-group)、[Toast](./project-components/toast)。
